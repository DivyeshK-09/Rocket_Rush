# Rocket Rush

## Overview
A Python-based arcade-style game inspired by Flappy Bird mechanics where a rocket navigates through obstacles.

## Objective
- Build a simple yet engaging game
- Understand game loops and physics

## Features
- Real-time gameplay
- Obstacle generation
- Score tracking
- Collision detection

## Tech Stack
- Python
- Pygame

## Architecture
1. Game loop
2. User input handling
3. Physics (gravity, movement)
4. Collision detection
5. Score update

## Results
- Smooth gameplay
- Responsive controls

## Challenges
- Collision accuracy
- Frame rate consistency

## Future Improvements
- Add sound effects
- Difficulty scaling
- Mobile version

python main.py
